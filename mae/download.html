My Avatar Editor for Your Desktop

To install My Avatar Editor for your desktop

  1. Download Adobe® AIR™ Runtime for your Windows/macOS PC
  2. Download My Avatar Editor Installer
  3. Follow this tutorial to get the My Avatar Editor Desktop applicatiom working!

Install My Avatar Editor for your desktop by clicking on the install button on the right. Alternatively, you can also download the AIR installer files directly.

Note: Adobe AIR must be installed in order to use the AIR installer.

The desktop version of My Avatar Editor includes all of the features of the editor on helloT's My Avatar Editor website revival, as well as the following additional features:

  • Associated file types for files of the extension .mae and .mii (opening these files will cause them to be opened in the editor)
  • Support for drag and drop loading of avatar files of any supported format (.mii, .mae, .rcd and .xml)

Requirements:

  • Adobe AIR 1.5 or later

My Avatar Editor for Your Website

My Avatar Editor is made of 2 SWF (Flash file format) files, myavatareditor.swf and myavatarcharacter.swf. Both are published with a target of Flash Player 10 so users will need to have at least that version of Flash Player installed in order to view them.

  • My Avatar Editor SWFs (zip)
    (ver 1.0.4.0)

More information about working with these files can be found in the Developers Section.
